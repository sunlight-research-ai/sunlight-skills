# Code of Conduct

We want Sunlight Skills to be a useful, welcoming place for people building with agentic workflows.

Participants are expected to:

- be respectful and constructive
- assume good intent while still being clear about technical issues
- avoid harassment, personal attacks, and discriminatory language
- focus feedback on the work and its usefulness to users

Maintainers may remove issues, pull requests, comments, or contributors that violate these expectations.
